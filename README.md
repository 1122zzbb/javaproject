# Java 电商展示系统

## 项目简介

基于 Java Swing 开发的桌面端电商商品展示系统，模拟电商平台商品浏览界面。

## 技术栈

- **语言**: Java
- **GUI 框架**: Swing
- **开发环境**: IntelliJ IDEA

## 功能特性

- 商品网格展示（5行2列布局）
- 商品图片展示
- 商品信息展示（价格、销量、标签等）
- 热卖/秒杀/新品标签
- 点击商品查看详情
- 购物车功能提示

